# Supermarket_admin
ระบบบันทึกรายการสินค้าของ supermarket ผู้ใช้สามารถเพิ่ม แก้ไข และลบ รายการสินค้าได้

## วิธีการติดตั้ง Package ที่เกี่ยวข้อง
$ npm i express mysql2 ejs multer body-parser nodemon

## วิธีการ Run ระบบ
$ npm run start

## เปิด Database ควบคู่ไปด้วยทุกครั้ง
แก้ไขข้อมูล Database ได้ที่ไฟล์ app.js
